# Rusty
This a calculator based on the programming language **Rust** 
If you dont know what **Rust** is then you need to know what it is
### **Rust** is a programming language that disallows the following:
- Memory Bugs.
- Memory leaks.
- Null pointer reference. 
### **Rust** includes the following:
- Mutable and immutable variables so that you dont accidently change a variable you didn't want to vice versa.
- C/C++ like syntax, The syntax is fairly easy to learn if you have prior experience with C/C++.
- Very Easy debugging, The compiler is so well designed that it tells you where the error is and **how** to fix it.
- Can talk with other languages like C/C++
- It has the benefits of Memory safety, C/C++'s performace, what else do you need?
